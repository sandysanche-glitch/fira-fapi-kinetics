# FIRA FAPI crystallization kinetics and spatial fingerprints

This repository contains the manuscript-oriented analysis code, processed CSV outputs, and figure-generation assets used to study crystallization kinetics and spatial fingerprints in FIRA-processed FAPI and FAPI--TEMPO perovskite films.

## Current status
This is a curated first-pass repository build created from a larger working directory. It separates:
- manuscript-critical scripts and processed outputs
- archived exploratory scripts and legacy figure assets
- documentation and starter manifests for reproducibility

## Repository structure
- `src/`: core analysis modules grouped by workflow stage
- `data/processed/`: processed CSV outputs used in manuscript analysis
- `results/`: final or near-final exported figures
- `docs/`: starter manifests and workflow notes
- `archive/`: exploratory, legacy, or uncertain files retained for traceability

## Important note
Some manuscript figures were assembled from multiple panel-level scripts and then combined externally. This repository therefore preserves a figure-oriented workflow rather than forcing a single-script-per-figure model.
